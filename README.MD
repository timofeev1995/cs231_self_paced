#### **Assignments info**

The course consists of 3 assignments. For all of them assignee have to fill out .ipynb notebooks at the root (in each assignment) and write some code in given .py modules - classes and functions.

#### **Other activities**

In the original course there are [final projects](http://cs231n.stanford.edu/project.html) examples for Stanford students.
